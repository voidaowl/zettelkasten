# Multidimensional array
Array can have items that are also arrays. We can use it for multidimensional arrays, for example to store matrices:
```JavaScript title:example.js
let matrix = [
	[1, 2, 3],
	[4, 5, 6],
	[7, 8, 9]
];

alert( matrix[0][1] ); // 2
```
