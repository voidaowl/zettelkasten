# map
Syntax: `arr.map(function(item, index, arr) {...});`.
Calls the function for each element of the array and returns the array of results.

```JavaScript title:example.js
let lengths = ["Bilbo", "Gandalf", "Nazgul"].map(item => item.length);
alert(lengths); // 5,7,6
```
