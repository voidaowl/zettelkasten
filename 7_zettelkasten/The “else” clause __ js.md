# The “else” clause
The `if` statement may contain an optional `else` block. It executes when the condition is falsy.

```JavaScript title:example.js
let year = prompt("What's the current year?")

if (year == 2025) {
	alert('True!');
} else {
	alert('Nope! Try again.');
}
```
